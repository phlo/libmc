libmc is a python module providing a collection of tools for KV Model Checkin and implements the following concepts presented in the lecture.

Completeness and Determinism¶

libmc.LTS.isComplete() Completeness (p21): True if LTS is complete.
libmc.LTS.isDeterministic() Determinism (p21): True if LTS is deterministic.

Product Automata and Sub-Set Construction¶

libmc.LTS.product(other) Create product LTS.
libmc.LTS.power() Create power LTS.

Labelled Transition Systems¶

libmc.LTS(S, I, Σ, T) Labelled Transition System (p31).

Simulation and Bisimulation¶

libmc.LTS.simulates(other, τ=[]) Simulation of LTS (p39).
libmc.maximumSimulation(A1, A2, R0, τ=[]) Constructs the maximum simulation relation A1 ≲ A2 (p36).
libmc.LTS.bisimulates(other, τ=[]) Bisimulation of LTS (p43).
libmc.maximumBisimulation(A1, A2, R0, τ=[]) Constructs the maximum bisimulation relation A1 ≈ A2 (p43).
