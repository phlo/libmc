Overview¶

libmc is a python module providing a collection of tools for KV Model Checking and implements the following concepts presented in the lecture.

Completeness and Determinism¶

libmc.LTS.isComplete Completeness (p21): True if LTS is complete.
libmc.LTS.isDeterministic Determinism (p21): True if LTS is deterministic.

Product Automata and Sub-Set Construction¶

libmc.FA.product Create product automaton (p20).
libmc.LTS.product Create product LTS.
libmc.FA.power Create power automaton (p22).
libmc.LTS.power Create power LTS.

Labelled Transition Systems¶

libmc.LTS Labelled Transition System (p31).

Asynchronous Composition¶

libmc.asynchronousComposition Asynchronous composition of LTS through interleaving (p84).

Simulation and Bisimulation¶

libmc.LTS.simulates Simulation of LTS (p39).
libmc.maximumSimulation Constructs the maximum simulation relation A1 ≲ A2 (p36).
libmc.LTS.bisimulates Bisimulation of LTS (p43).
libmc.maximumBisimulation Constructs the maximum bisimulation relation A1 ≈ A2 (p43).

Traces¶

libmc.LTS.trace Tries to compute all traces to the target state (DFS).

Finite Automata¶

libmc.FA Finite Automata (p18).

Acceptance¶

libmc.FA.accepts Test acceptance of a given word.

Complement¶

libmc.FA.complement Create complement automaton (p23).

Conformance¶

libmc.FA.conforms Conformance test (p24).

Minimization¶

libmc.FA.minimize Minimization of Deterministic Finite Automata (p44).

Breadth-First and Depth-First Search¶

libmc.bfs Generic breadth-first search.
libmc.dfs Generic depth-first search.

Tarjan’s Algorithm¶

libmc.tarjan Tarjan’s Algorithm (p110).

Graphviz DOT Language String Generation¶

libmc.fa2dot Returns graphical representation of given automaton (using Graphviz).
libmc.FA.toDot Return Graphviz dot language string.
libmc.LTS.toDot Return LTS as Graphviz dot language string.
